# HelpDesk Acadêmico 🎫💻

<p>

    Uma aplicação web interativa para cadastro e acompanhamento de chamados técnicos, desenvolvida com foco em manipulação do DOM e interatividade utilizando JavaScript, HTML e CSS.
</p>


## 📌 Funcionalidades

<p>

    - Cadastro de chamados técnicos
    
    - Validação de campos obrigatórios
    
    - Exibição dos chamados em cards dinâmicos
    
    - Registro automático de data e hora
    
    - Contador total de chamados
    
    - Indicadores por prioridade
    
    - Indicadores por tipo de problema
    
    - Alternância entre modo claro e modo escuro
    
    - Alteração dinâmica de estilos via JavaScript
    
    - Interface responsiva com Bootstrap
</p>

## 📚 Conceitos aplicados

<p>
    Este projeto utiliza conceitos fundamentais de desenvolvimento web front-end:
    
    Manipulação do DOM
    
    Eventos JavaScript

    Atualização em tempo real da interface
</p>

## 🎯 Objetivo do projeto

<p>

    Este projeto foi desenvolvido como atividade acadêmica com o objetivo de praticar JavaScript, HTML e CSS no desenvolvimento de interfaces dinâmicas e interativas, juntamente aos templates do Bootstrap.
</p>
